FreeBSD コミッターのためのリソース

コミッターのためのガイド

すべての新規コミッターを対象とした、
技術的な詳細とポリシーの詳細を含む包括的な文書です。 前者については、
Subversion VCS, Bugzilla バグ追跡システム、および SSH Quick-Start Guide
章をご覧ください。

新しいコミッターが知るべきほとんどすべてのことが記述されています。
より多くのことを知るために、新しい貢献者のための FreeBSD
ドキュメンテーションプロジェクト入門や Ports ガイドも参照してください。

新しい貢献者のための FreeBSD ドキュメンテーションプロジェクト入門

この入門書には、FreeBSD
ドキュメンテーションプロジェクトへの貢献を始めるために知っておくべきすべてのことが書いてあります。
あなたが使うであろう (必須や推奨される) ツールやソフトウェアから、
ドキュメンテーションプロジェクトの背後にある哲学まで記述しています。

port 作成者のためのハンドブック

FreeBSD ports 作成者のためのガイド。公式的な __FreeBSD_version
の値のテーブル があります。

FreeBSD Web ページの構築方法

この文書は、FreeBSD Web ページを手動で SVN
リポジトリから構築して更新する方法を解説しています。
