获取 FreeBSD

选择一种架构

大多数 FreeBSD 用户拥有 amd64 或 aarch64 架构的硬件，
这些硬件拥有很好的支持。

现代个人电脑使用 amd64 架构， 包括那些使用 Intel® 品牌处理器的电脑。
对于嵌入式设备和单板机， 例如 Raspberry Pi 3 和 4、 ESPRESSObin，
以及许多全志科技和瑞芯微电子的开发板， 可以使用 aarch64 SD 卡映像。

其他用户请参阅 FreeBSD 支持的平台的 完整列表。

选择一个映像

安装 FreeBSD 有许多种选择， 包括从 CD (disc1)、 DVD (dvd1)、
网络安装映像 (bootonly) 和 USB 映像中启动 FreeBSD 安装程序。 对于最近的
FreeBSD 版本， 虚拟机映像和用于嵌入式平台的 SD 卡映像也可供下载。

FreeBSD 部署数据

尽管 FreeBSD 并不收集部署数据， 获取统计信息是十分重要的。 请考虑安装
sysutils/bsdstats 包， 这将收集硬件和软件的统计数据，
以帮助开发者理解如何更好地进行开发。 收集的信息可在 bsdstats.org
网站查阅。

如果您对一个完全实验性的 FreeBSD-CURRENT (&rel.head;-CURRENT) 发行版
快照 感兴趣， 请参阅 FreeBSD 发行版快照 页。
快照仅供开发者和前沿测试者使用。 欲了解关于过去、
现在和未来发行版的更多信息， 请参阅 发行版本信息 页。

powerpc64, powerpc64le, powerpcspe, armv6, armv7, aarch64, and riscv64.

Installer and SD card images are available for amd64, i386, powerpc,
powerpc64, powerpc64le, powerpcspe, armv6, armv7, aarch64, and riscv64.

Installer and SD card images are available for amd64, i386, powerpc,
powerpc64, powerpcspe, sparc64, armv6, armv7, and aarch64.

安装 FreeBSD

安装 FreeBSD 有许多种选择， 包括从 CD-ROM、 DVD、 USB 存储棒、 匿名
FTP、 HTTP 以及 NFS 安装。 请在下载完整的 FreeBSD 发行版之前阅读
FreeBSD安装说明 。

购买 FreeBSD 安装盘

FreeBSD 可以在 FreeBSD Mall 或其它的 CD-ROM 和 DVD 发行商 处取得 CD-ROM
或 DVD。

过去的发行版

若要下载过去的发行版， 请到 FTP 存档 下载。

FreeBSD 衍生的操作系统发行版

FreeBSD 被广泛用于其它商业和开源操作系统的组成部分。
以下的项目被广泛使用并尤其使 FreeBSD 用户感兴趣。

-   FreeNAS 是一个基于 FreeBSD 的开源存储平台， 它支持在 Windows、 Apple
    和类 UNIX 系统之间分享文件。

-   GhostBSD 源自 FreeBSD， 它使用 GTK 环境在现代 BSD
    平台上提供漂亮的外观和舒适的体验， 提供自然和原生的 UNIX 工作环境。

-   MidnightBSD 源自 BSD， 为桌面用户开发。
    它包括您期望用于日常任务的所有软件： 邮件、 网页浏览、 文字处理、
    游戏等等。

-   NomadBSD NomadBSD 是一个基于 FreeBSD 的用于 USB 闪存驱动器的系统。
    它拥有自动硬件检测和设置， 被配置为开箱即用的桌面系统，
    但也可用于数据恢复、 教育目的或测试 FreeBSD 的硬件兼容性。

-   pfSense 是一个基于 FreeBSD 的定制发行版，
    为防火墙和路由器使用专门定制。
