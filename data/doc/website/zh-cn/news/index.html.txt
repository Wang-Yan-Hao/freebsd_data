FreeBSD 新闻

[FreeBSD 新闻]

本地新闻

-   最新消息： 新的发行版， 驱动程序， committers， 安全公告，
    以及其他新闻。

-   新闻发布： 来自 FreeBSD project 以及其他相关组织的新闻。

-   报章新闻： 在传统报章上的 FreeBSD 相关消息。

-   进度报告： FreeBSD 开发进度报告。

其他站点

-   BSD DevCenter： ONLamp.com/O’Reilly 网络的 BSD 文章、 新闻、 教程，
    以及一般的社区信息的集散地。

-   BSDForums.org： 一个关于 FreeBSD 和其他 BSD
    的活跃的在线论坛和专属新闻社区网站。

-   BSD Freak： 一个专门发布用户视角的 BSD 相关文章和教程的站点。

-   BSDnews： 关于各种 BSD 的在线杂志。

-   BSDPortal.org： 这个网站收集了来自其他关于 BSD 网站上最重要的新闻，
    提供了关于所有 BSD 操作系统的最新进展。

-   DaemonNews (BSDnews.com)： 每月连载的文章以及关于 BSD 的各种讨论。

-   FreeBSD Diary： 一个人关于他的 FreeBSD 经历的日记。

-   Kerneltrap： 关于 BSD 和 Linux 内核的最新动态。

-   OSNews： 关于 Linux， BSD 以及其他各类操作系统的最新消息。

-   Slashdot 的 BSD 部分： 同时也有关于其他 BSD 的新闻和讨论，
    并不仅限于 FreeBSD.
