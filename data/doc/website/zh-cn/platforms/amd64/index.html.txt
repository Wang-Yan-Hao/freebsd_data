FreeBSD/amd64 Project

[BSD Daemon swinging a sledge hammer]

本页包含的信息是关于如何将 FreeBSD 移植到http://www.amd.com/[AMD] 的
AMD64 和 Intel® EM64T (Extended Memory 64-bit Technology)架构。 AMD64
架构以前常被称为 x86-64 或者“Hammer(铁锤)”。 Intel EM64T
架构以前常被称为 IA-32e，Clackamas 技术和 Yamhell。

The AMD Opteron™、 AMD Athlon™ 64、 AMD Turion™ 64 和较新的 AMD Sempron™
处理器使用 AMD64 架构。

状态：

当前 FreeBSD/amd64 可运行在 64
位多用户模式，单处理器模式和多处理器模式。

AMD64 平台目前是 Tier 1 的 FreeBSD 平台。
