Projet Java® pour FreeBSD : FAQ

Contenu

1.  Existe-t’il un JDK® 1.3 pour FreeBSD ?

2.  Puis-je utiliser le port Linux du projet Blackdown ?

Les réponses

1.  Existe-t’il un JDK® 1.3 pour FreeBSD ? Oui. Vous pouvez compiler un
    JDK FreeBSD natif à partir des sources, ou télécharger des binaires
    précompilés. Voir la page principale FreeBSD Java pour plus
    d’informations.

2.  Puis-je utiliser le port Linux du projet Blackdown ?

Plusieurs personnes ont réussi. Essayez les instructions suivantes :
Télécharger l’archive et l’extraire dans un répertoire. Appliquer
ensuite le diff suivant :

    --- bin/.java_wrapper.dist      Thu Jun  3 01:14:34 1999

    +    link=`/compat/linux/usr/bin/expr "$ls" : '.*-> \(.*\)$'`
    +    if /compat/linux/usr/bin/expr "$link" : '/' > /dev/null; then

    else
    PRG="`/usr/bin/dirname $PRG`/$link"
