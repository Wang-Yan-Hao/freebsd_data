Projet FreeBSD/pc98

FreeBSD/pc98 est un port de FreeBSD dont l’objectif est de fonctionner
sur l’architecture NEC PC-98x1 (pc98). Le but du projet est de faire en
sorte que FreeBSD/pc98 fonctionne aussi bien que sur les autres
architectures. La majorité du code source noyau est déjà inclu dans
l’arbre des sources FreeBSD et la plupart des programmes utilisateurs
compilés à partir des sources fonctionnent parfaitement.
