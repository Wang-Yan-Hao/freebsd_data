Proyecto FreeBSD/i386

Introducción

Esta página contiene información acerca de la plataforma FreeBSD/i386.
Esta arquitectura soporta una extensa variedad de CPU ya que la mayoría
de los usuarios compran este tipo de máquinas. Si se tiene una máquina
de una compañía de PC (tal como Dell™, HP o eMachines®) lo más probable
es que tenga una CPU AMD™ o Intel®, que pertenecen a esta categoría.

Últimas noticias

-   FreeBSD 6.0 (y versiones posteriores) no soporta la CPU original
    Intel 80386; estos sistemas tienen más de siete añ y sólo FreeBSD
    5.X (y versiones anteriores) las soportan. El kernel GENERIC incluye
    soporte para la CPU 80386 sólamente en FreeBSD 4.X y versiones
    anteriores.
