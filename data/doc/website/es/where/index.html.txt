Obtener FreeBSD

Elegir una arquitectura

La mayoría de los usuarios de FreeBSD tendrán hardware para las
arquitecturas amd64 o aarch64, las cuales están bien soportadas.

Lor ordenadores modernos utilizan la arquitectura amd64, incluidas
aquellos con procesadores de la marca Intel ®. Los dispositivos
embebidos y los single-board computers (SBCs), como las Raspberry Pi 3 y
4, ESPRESSObin, así como Allwinner y Rockchip, utilizan la imagen de la
tarjeta SD aarch64.

El resto de usuarios/as deben consultar la lista completa de plataformas
soportadas por FreeBSD.

Elegir una imagen

El instalador de FreeBSD se puede descargar en varios formatos
diferentes, incluidos CD (disco 1), DVD (dvd1) e imágenes ISO para la
instalación en red (bootonly), así como imágenes de memoria USB normal y
mini. Las versiones reciendes de FreeBSD también se pueden descargar
como imagenes precompiladas para máquinas virtuales y como imagenes para
tarjetas SD para plataformas embebidas.

Estadísticas de implementación de FreeBSD

Si bien FreeBSD no recopila estadísticas de instalación, es esencial
tener información estadística disponible. Considere instalar el paquete
sysutils/bsdstats, que recopila estadísticas de hardware y software, el
cual ayuda a los desarrolladores/as a comprender cómo enfocar mejor sus
esfuerzos. La información recogida está disponible en el sitio web
bsdstats.org.

Snapshots de desarrollo

Si está interesado/a en una versión snapshot con puramente experimental
de FreeBSD-CURRENT (también conocido como 14.0-CURRENT), dirigida
únicamente a desarrolladores/as y usuarios/as experimentados, consulte
la página de lanzamientos de las Snapshot de FreeBSD. Para obtener más
información sobre lanzamientos pasados, presentes y futuros, visite la
página de información sobre lanzamientos.

FreeBSD 14.0-CURRENT

Las imágenes del instalador y de la tarjeta SD están disponibles para
amd64, i386, powerpc, powerpc64, powerpc64le, powerpcspe, armv6, armv7,
aarch64, y riscv64.

Las imágenes para máquinas virtuales están disponibles para amd64, i386,
aarch64, and riscv64.

FreeBSD 13.1-STABLE

Las imágenes del instalador y de la tarjeta SD están disponibles para
amd64, i386, powerpc, powerpc64, powerpc64le, powerpcspe, armv6, armv7,
aarch64, y riscv64.

Las imágenes para máquinas virtuales están disponibles para amd64, i386,
aarch64, and riscv64.

FreeBSD 12.3-STABLE

Las imágenes del instalador y de la tarjeta SD están disponibles para
amd64, i386, powerpc, powerpc64, powerpcspe, sparc64, armv6, armv7, y
aarch64.

Las imágenes para máquinas virtuales están disponibles para amd64 and
i386.

Si planea obtener FreeBSD a través de HTTP o FTP, consulte la lista de
sitios espejo en el Handbook para ver si hay un sitio más cercano.

Instalar FreeBSD

Hay muchas opciones para instalar FreeBSD, incluida la instalación desde
CD-ROM, DVD, memoria USB o incluso directamente mediante FTP, HTTP o
NFS. Lea la guía de instalación de FreeBSD antes de descargar toda la
distribución de FreeBSD.

Comprar CD-ROM o DVD de FreeBSD

FreeBSD se puede adquirir en CD-ROM o DVD en FreeBSD Mall, o en uno de
los otros vendedores/as de CD-ROM y DVD.

Versiones anteriores

Para descargar versiones anteriores, visite el archivo FTP.

Sistemas operativos derivados de FreeBSD

FreeBSD se usa ampliamente como componente básico para otros sistemas
operativos comerciales y de código abierto. Los siguientes proyectos son
utilizados ampliamente y de interés particular para los usuarios/as de
FreeBSD.

-   TrueNAS es un software de almacenamiento conectado en red, Network
    Attached Storage (NAS) que comparte y protege los datos de las
    amenazas actuales, como el ransomware y el malware. TrueNAS facilita
    que los usuarios/as y los dispositivos cliente accedan a los datos
    compartidos a través de prácticamente cualquier protocolo de uso
    compartido.

-   GhostBSD se deriva de FreeBSD, GhostBSD utiliza el entorno GTK para
    brindar una apariencia hermosa y una experiencia cómoda en una
    moderna plataforma BSD que ofrece un entorno de trabajo UNIX® nativo
    y natural.

-   MidnightBSD es un sistema operativo derivado de BSD desarrollado
    pensando en los usuarios/as de escritorio. Incluye todo el software
    que esperaría para sus tareas diarias: correo, navegación web,
    procesamiento de textos, juegos y mucho más.

-   NomadBSD es un sistema live para unidades flash USB, basado en
    FreeBSD. Junto con la detección y configuración automáticas de
    hardware, está configurado para usarse como un sistema de escritorio
    que funciona out of the box, también se puede usar para la
    recuperación de datos, con fines educativos o para probar la
    compatibilidad del hardware de FreeBSD.

-   pfSense es una distribución personalizada basada en FreeBSD adaptada
    para su uso como firewall y router.
