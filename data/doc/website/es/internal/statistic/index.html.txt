Estadísticas de FreeBSD

Estadísticas Web

Analog

analog(1) tool.

Las estadísticas de uso de este servidor actualizadas diariamente con la
herramienta analog(1).

Urchin

El paquete estadístico Urchin es usado para crear estas estadísticas
sobre el uso del servidor web.

Uso de releases de FreeBSD

Una muestra del uso de la release actual de FreeBSD está disponible en
http://www.freebsd.org/statistic/release_usage/images/.

El contador de FreeBSD

La página The FreeBSD Counter Page es el comienzo de un proyecto con el
que se intenta determinar la base instalada a nivel mundial de usuarios
de FreeBSD.
