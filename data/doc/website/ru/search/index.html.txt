Архивы списков рассылки

Переиндексация архива списков рассылки теперь происходит еженедельно!

В качестве альтернативы для чтения списков рассылки может быть
использован Mailman/Pipermail архив (учтите, что этот архив содержит
сообщения только с марта 2003 года).

Списки рассылки (а также много чего еще) также архивируются на сервере
GeoCrawler.
