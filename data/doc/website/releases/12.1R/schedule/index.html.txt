FreeBSD 12.1 Release Process

Introduction

This is the release schedule for FreeBSD 12.1. For more information
about the release engineering process, please see the Release
Engineering section of the web site.

General discussions about the pending release and known issues should be
sent to the public freebsd-stable mailing list. MFC requests are no
longer being accepted.
