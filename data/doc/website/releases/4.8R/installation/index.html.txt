FreeBSD 4.8-RELEASE Installation Notes

The installation notes for FreeBSD are customized for different
platforms, as the procedures for installing FreeBSD are highly dependent
on the hardware platform.

Installation notes for FreeBSD 4.8-RELEASE are available for the
following platforms:

-   alpha
-   i386

A list of all platforms currently under development can be found on the
Supported Platforms page.
