Das X Window-System

-   Das X.Org-Projekt bietet Anwendern verschiedener Intel-basierter
    UNIX®-Systeme, darunter auch FreeBSD, ein exzellentes X
    Window-System.

-   Das WINE-Projekt ermöglicht es, verschiedene Microsoft
    Windows®-Programme auf Intel-basierten UNIX-Systemen wie FreeBSD,
    NetBSD oder Linux auszuführen.

Hardware

-   Die comp.answers pc-hardware-faq enthalten wertvolle Informationen
    für Personen, die einen eigenen Rechner bauen wollen.

-   Aad Offerman’s Chip List - Referenzmaterial für in PC-Clones
    verwendete Computerchips.

Betriebssystem-Projekte

-   NetBSD ist ein weiteres freies auf 4.4BSD Lite basierendes
    Betriebssystem, das auf vielen verschiedenen Architekturen lauffähig
    ist.

-   OpenBSD stammt ebenfalls von 4.4BSD ab, legt den Schwerpunkt aber
    auf die Systemsicherheit.

-   Auch Linux ist ein freies UNIX®-artiges System.

-   Lites ist eine auf 4.4BSD Lite basierende Server- und
    Emulationsbibliothek, die freie UNIX-Fähigkeiten auf einem
    Mach-basierten System bietet.

-   Das GNU HURD-Projekt ist ein weiterer Versuch zur Entwicklung eines
    freien UNIX®-artigen Betriebssystems.

-   DragonFly BSD basiert zwar auf FreeBSD 4.X, die Entwicklungsziele
    unterscheiden sich aber von denen von FreeBSD.

-   illumos ist ein Fork von OpenSolaris, welches wiederum seine Wurzeln
    in einem als SunOS bekannten BSD-Derivat hat.
